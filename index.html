Online timing diagram editor

WaveDrom 在线可视化编辑器

用图形化方式创建、修改和导出时序波形。主页保持简洁入口，编辑器负责完整的行、周期和标签编辑。

说明文档

教程将单独整理成独立页面。当前版本可以通过“快速开始”进入编辑器，并在左侧“示例”中载入 WaveDrom 官方教程示例。

许可

本项目使用 WaveDrom JS，并遵循 WaveDrom 的 MIT License。你创建和导出的波形内容归你所有。

反馈与支持

遇到问题可以通过“作者/支持”进入 GitHub Issues 反馈；如果这个工具节省了你的时间，也可以扫码支持项目维护。
